✈️ Flight Visualization & Air Traffic Management System

This is a Flight Visualization and Air Traffic Management System built using React and Python. It displays real-time flight data, runway allocation, and congestion levels through an interactive dashboard.

🚀 Getting Started

Note: Make sure you have Node.js and Python installed before proceeding.

📦 Step 1: Install Dependencies

Navigate to the project folders and install required packages.

Frontend (React)
cd frontend
npm install
Backend (Python)
cd backend
pip install -r requirements.txt
▶️ Step 2: Run the Application
Start Backend Server
cd backend
python app.py
Start Frontend (React App)

Open a new terminal:

cd frontend
npm start
🌐 Step 3: Open the App

After running both servers, open your browser and go to:

http://localhost:3000

You should now see the Flight Visualization Dashboard running.

✏️ Step 4: Modify the Project
Edit frontend files inside src/
Update backend logic in app.py
Changes will reflect automatically in the browser
